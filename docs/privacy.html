개인정보처리방침

최종 업데이트: 2026년 5월 16일

TickLab (이하 "본 앱") 은 사용자의 개인정보 보호를 매우 중요하게 생각합니다. 본 방침은 본 앱이 수집하는 정보의 종류, 사용 목적, 보관 방식, 사용자의 권리를 설명합니다.

1. 수집하는 정보

본 앱은 개인정보를 외부로 전송하거나 수집하지 않습니다. 모든 데이터는 사용자의 기기 내부에만 저장됩니다.

  • 마이크 입력: 측정 중 실시간으로 시계 박동을 분석하는 데만 사용됩니다. 오디오는 즉시 폐기되며 녹음·저장·전송되지 않습니다.
  • 사진: 사용자가 직접 선택한 시계 사진은 EXIF 위치 정보 제거 후 기기 내 SwiftData 에 저장됩니다.
  • 측정 데이터·일기·서비스 기록: 모두 기기 내부의 로컬 데이터베이스에만 저장됩니다.
  • 자기장 센서: 시계 자성화 위험 확인용. 측정값은 기기 내부에서만 처리됩니다.

2. 외부 전송 데이터

다음 항목만 사용자 동의 하에 외부로 전송될 수 있습니다.

  • 브랜드 리그 (선택): 사용자가 등록한 시계 브랜드명과 착용 빈도가 익명으로 글로벌 랭킹 서버에 전송됩니다. 개인 식별 정보·시리얼·사진은 일절 전송되지 않습니다.
  • 무브먼트 DB 업데이트 (선택): 설정에서 활성화한 경우에만 최신 무브먼트 데이터를 다운로드합니다.

3. 결제 정보

TickLab Pro 연간 구독 (9.99 USD/년) 결제는 Apple App Store 가 모두 처리합니다. 본 앱은 사용자의 결제 카드 정보·청구 주소·Apple ID 에 직접 접근하지 않습니다. 자세한 내용은 Apple 개인정보처리방침 을 참조하세요.

4. AI 진단

측정 결과 AI 진단은 Apple Intelligence (FoundationModels) 를 사용해 기기 내부에서 처리됩니다. 측정 데이터는 Apple 또는 OpenAI 등 외부 서버로 전송되지 않습니다. Apple Intelligence 미지원 디바이스에서는 규칙 기반 fallback 이 동작합니다.

5. 데이터 보관 및 삭제

모든 데이터는 기기 내부 SwiftData 에 저장됩니다. 앱을 삭제하면 모든 데이터가 함께 영구 삭제됩니다. 설정 → 데이터 관리에서 개별 시계·측정·일기·서비스 기록을 언제든 삭제할 수 있습니다.

6. 권한

본 앱은 다음 권한을 요청할 수 있습니다.

  • 마이크 — 시계 박동 측정
  • 카메라 — 시계 사진 촬영
  • 사진 라이브러리 — 기존 사진 선택 / 공유 카드 저장
  • 알림 — 일기·정비·랜덤 시계 추천 알림
  • Face ID — 앱 잠금 (선택)

7. 아동 보호

본 앱은 13세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.

8. 방침 변경

본 방침은 필요 시 업데이트될 수 있으며, 변경 사항은 본 페이지를 통해 공지됩니다.
