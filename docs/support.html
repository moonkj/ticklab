자주 묻는 질문

측정 결과가 평소보다 큰 오차로 나와요

가장 흔한 원인은 동력 부족입니다. 시계를 풀와인딩 (수동 30~40 회전 또는 자동 시계는 1~2분 흔들기) 한 후 5분 안정화 시간을 두고 재측정해 보세요. 측정 자세 (다이얼 업·다이얼 다운·크라운 업·다운·좌·우) 가 picker 선택과 일치하는지도 확인하세요.

amplitude (진폭) 가 표시되지 않아요

일부 무브먼트 (사파이어 케이스백, 코악시얼 이스케이프먼트 등) 는 마이크 기반 진폭 측정 정확도가 낮아 의도적으로 표시하지 않습니다. rate (일오차) 와 beat error 는 정상적으로 측정됩니다.

Pro 결제 후 기능이 바로 풀리지 않아요

설정 → 계정 카드 탭 → "이전 구매 복원" 을 눌러보세요. 같은 Apple ID 로 App Store 에 로그인되어 있는지 확인하세요. 그래도 문제가 있으면 support@ticklab.app 으로 영수증 이메일과 함께 연락주세요.

구독 취소 방법

iPhone 설정 앱 → [본인 Apple ID] → 구독 에서 TickLab Pro 선택 → 취소. 또는 앱 설정 → 계정 카드 탭하면 구독 관리 시트가 열립니다.

측정 도중 화면이 꺼져요

설정 → 측정 섹션 → "측정 중 화면 항상 켜기" 가 켜져 있는지 확인하세요. 기본값은 ON 입니다.

일기·시계 데이터가 사라졌어요

모든 데이터는 기기 내부에 저장되며 iCloud 백업은 아직 지원되지 않습니다. 디바이스 변경·앱 삭제 시 데이터가 함께 삭제됩니다. 데이터 export 는 시계 디테일 → 메뉴 → "내보내기 (CSV)" 로 가능합니다.

AI 진단이 작동하지 않아요

AI 진단은 Apple Intelligence 지원 디바이스 (iPhone 15 Pro 이상, iOS 18.1+) 에서만 동작합니다. 미지원 디바이스에서는 규칙 기반 fallback 이 자동 사용됩니다. Free 사용자는 시계당 3회 trial 후 fallback 으로 전환됩니다.

측정 정확도가 표시되는 기준은?

측정 결과에는 A/B/C/F grade 가 부여됩니다. A·B 는 신뢰 가능 범위, C·F 는 재측정을 권장합니다. 신뢰도는 신호 대 잡음비 (SNR), 측정 시간, BPH lock 품질, beat error 종합으로 결정됩니다.
